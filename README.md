# TechX Ecommerce (Portfolio Project)

A fictional ecommerce storefront concept built with Next.js 15, Shadcn UI, and Tailwind CSS.  
Created as part of my web development portfolio.

## Features

- Product listing and dynamic product pages
- Cart system with React state management
- Responsive UI with Shadcn components
- Modern file-based routing (Next.js App Router)

## Tech Stack

- Next.js 15
- Shadcn UI
- Tailwind CSS
- TypeScript

## Running the Project

````npm run dev
